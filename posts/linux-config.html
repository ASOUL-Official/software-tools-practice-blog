Linux Config

Linux 环境配置

本页用于整理 Ubuntu 22.04 WSL2、系统 profile、用户环境变量和账户权限配置，对应评分项“Linux环境配置”和“Linux账户配置”。

目标 网络与系统 账户与权限 证据表

实验目标

完成一套可用于部署的 Linux 环境配置，使它具备可访问网络地址、清晰主机名、独立开发用户、可追踪的环境变量配置和合理的文件权限。

网络与系统配置

hostnamectl
ip addr
echo $PATH
cat ~/.bashrc
sudo cat /etc/profile

建议使用 NAT + Host-Only 双网卡。Host-Only 地址固定后，宿主机可以稳定通过 SSH 连接虚拟机。

账户与权限配置

sudo adduser dev
sudo usermod -aG sudo dev
id dev
groups dev
ls -la /home/dev
chmod 755 /home/dev/practice-blog

账户配置要体现“普通用户日常开发，sudo 只在系统管理时使用”的原则。
